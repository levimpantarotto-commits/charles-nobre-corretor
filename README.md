# Charles R. Nobre | Consultoria Imobiliária

Este projeto é uma plataforma premium de consultoria imobiliária focada no Litoral Sul de Santa Catarina (Imbituba, Garopaba, Imaruí). O foco é oferecer uma experiência de curadoria de luxo, conectando pessoas ao seu próximo estilo de vida.

## 🚀 Como Iniciar

Este é um projeto Next.js. Para rodar localmente:

```bash
npm install
npm run dev
```

Acesse: [http://localhost:3000](http://localhost:3000)

## 📂 Organização do Projeto

O projeto foi organizado para facilitar a manutenção e o crescimento:

-   `src/app/`: Rotas e páginas da aplicação.
-   `src/components/`: Componentes React reutilizáveis.
-   `docs/`: Manifestos, roteiros de marca e documentação técnica.
-   `scripts/`: Ferramentas de automação para processamento de logos, áudio e voz (Maestro).
-   `media/`: Arquivos de mídia gerados e assets temporários.
-   `public/`: Ativos estáticos acessíveis publicamente.

## 🎨 Identidade de Marca

A marca Charles R. Nobre preza pela elegância, silêncio e conexão com a natureza. 
Os detalhes de visão e roteiro podem ser encontrados em [docs/brand_manifest.md](docs/brand_manifest.md).
